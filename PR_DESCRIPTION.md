## トーナメントアプリ大規模改善: グループ制・モバイル対応・UI改善

### Summary
Matrix風トーナメント管理アプリに対する包括的な改善を行いました。

### 主な変更点

**グループ制トーナメントシステム（最大100人対応）**
- 22人以上の場合、自動でグループA〜E（各最大20人）に分割
- グループタブUIでグループ切り替え・進捗確認
- グループ勝者自動検出 → 全グループ完了後に決勝フェーズへ移行
- 管理者用「一括マッチメイク」ボタン

**トーナメント表の改善**
- 人数に応じた5段階のサイズ自動調整（2/4/8/16/32人以上）
- スマホ・タブレット完全対応（ビューポート幅に基づくスケーリング）
- シード権プレイヤーのBYE自動進出（勝利線の正しい描画）
- ウィンドウリサイズ対応（デバウンス付き）

**UI/UX改善**
- 「対戦確定」ボタン削除 → ルーレット完了後に自動で対戦確定
- 「タイトルに戻る」ボタンの遷移バグ修正（updateHudDisplay → updateHudVisibility）
- アイコン未登録ユーザーにシルエットアイコン表示（「?」→ SVG人形）
- 対戦カード抽選ボタンの視認性・自動発光改善
- 大会終了後のUI状態リセット改善

**マッチメイキングフロー改善**
- 奇数参加者のシード権を最後の1人に自動付与
- シード抽選モーダルを削除（フロー簡素化）

**データ管理**
- 大会参加者名簿をFirestore履歴に保存
- 管理者のみ大会終了時にCSV自動ダウンロード
- 締切後の参加者0人時に管理者がリセット可能
